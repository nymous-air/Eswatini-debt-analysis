# Debt Sustainability Analysis of Eswatini (2014–2023)

## Overview

This project analyzes long-term external debt trends in Eswatini using publicly available World Bank debt statistics data. 
The primary objective of the analysis is to evaluate debt sustainability by examining the relationship between long-term external debt and Gross National Income (GNI) over time.

Using R and RStudio, the project cleans and transforms raw Excel data into a usable analytical format and visualizes debt-to-GNI trends between 2014 and 2023.

---

## Objectives

* Clean and transform raw World Bank debt data
* Analyze long-term external debt trends in Eswatini
* Calculate the Debt-to-GNI ratio over time
* Visualize debt sustainability trends using ggplot2

## Data Source

World Bank Debt Statistics Dataset

The dataset contains economic and debt indicators for Eswatini from 2014–2023, including:

* Long-term external debt
* Gross National Income (GNI)
* Other external debt indicators

---

## Key Metric

The main metric used in this project is the Debt-to-GNI ratio:

Debt-to-GNI Ratio = (Long-term External Debt / Gross National Income) × 100

This metric helps evaluate whether debt is growing faster than the country's economic capacity.
